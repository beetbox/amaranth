Amaranth: A Command-Line AURA Client
====================================

This is an experimental command-line client for [AURA][], written in [Rust][]. The idea is to replicate the CLI of [beets][] but through a connection to a remote music library.

[aura]: https://github.com/beetbox/aura
[rust]: https://www.rust-lang.org
[beets]: http://beets.io
